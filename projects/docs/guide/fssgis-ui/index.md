# @fssgis/ui

![npm version](https://img.shields.io/npm/v/@fssgis/ui.svg)    ![downloads-badge](https://img.shields.io/npm/dm/@fssgis/ui.svg?color=orange)

## 介绍

Vue UI库

- use `TypeScript` and `TSX`
- build with `Rollup` 
- support vue3

## 基本使用

```bash
npm i @fssgis/ui
```

```js
import { /* ... */ } from '@fssgis/ui'
import '@fssgis/ui/dist/index.css'
```
